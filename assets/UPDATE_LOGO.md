# Hướng dẫn cập nhật Logo FLOW

## Bước 1: Đặt file logo mới

Đặt file logo FLOW vào thư mục `assets/` với tên:
- `flow-logo.png` (hoặc `flow-logo.svg`)

## Bước 2: Đổi tên file

Sau khi đặt file logo vào thư mục `assets/`, đổi tên thành:
- `favicon.png` (cho web favicon)
- `icon.png` (cho app icon - nếu cần)
- `adaptive-icon.png` (cho Android - nếu cần)

Hoặc cập nhật đường dẫn trong `app.json` và `README.md` để trỏ đến file logo mới.

## Bước 3: Kích thước khuyến nghị

- **favicon.png**: 64x64px hoặc 128x128px
- **icon.png**: 1024x1024px (cho iOS/Android)
- **adaptive-icon.png**: 1024x1024px (cho Android)

## Lưu ý

File logo sẽ được tự động sử dụng sau khi đặt vào thư mục `assets/` với tên đúng.
