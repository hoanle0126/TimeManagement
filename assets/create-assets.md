# Hướng dẫn tạo Assets

Để tạo các file hình ảnh cần thiết, bạn có thể:

1. **Favicon (favicon.png)**: 
   - Sử dụng file `favicon.svg` có sẵn
   - Convert sang PNG 32x32 hoặc 64x64
   - Hoặc sử dụng công cụ online: https://convertio.co/svg-png/

2. **App Icon (icon.png)**:
   - Kích thước: 1024x1024px
   - Nền trắng hoặc trong suốt
   - Icon checkmark màu xanh lá (#4CAF50)

3. **Adaptive Icon (adaptive-icon.png)**:
   - Kích thước: 1024x1024px
   - Nền trắng
   - Icon checkmark màu xanh lá (#4CAF50)

4. **Splash Screen (splash.png)**:
   - Kích thước: 1242x2436px (hoặc tỷ lệ tương tự)
   - Nền trắng
   - Logo TaskMaster ở giữa

Bạn có thể sử dụng các công cụ online như:
- https://www.favicon-generator.org/
- https://www.appicon.co/
- https://www.splash-screen-generator.com/
